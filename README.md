 Task 2: Sensor Data Simulation Overview

This repository contains the submission for Task 2 of the IoT Internship.

Objective

To simulate sensor data collection using an ESP32 and DHT22 sensor and display temperature and humidity readings.

Components Used

* ESP32 Development Board
* DHT22 Temperature and Humidity Sensor
* Wokwi IoT Simulator
* Arduino Programming Language

 Project Description

The DHT22 sensor was connected to an ESP32 microcontroller using the Wokwi simulator. The sensor collected temperature and humidity data, which was displayed through the Serial Monitor. The generated data was continuously monitored and logged during the simulation.

 Output

The simulation successfully displayed:

* Temperature readings
* Humidity readings
* Real-time sensor data monitoring

 Key Skills Learned

* Sensor Data Collection
* ESP32 Programming Basics
* Temperature and Humidity Monitoring
* Data Logging and Visualization
